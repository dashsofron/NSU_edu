Запуск с помощью выражения java -jar
 
lab1-test-epression.jar - сборка программы с выбранным мной тестовым выражением
Запускать:
java -jar .\lab1-test-epression.jar

lab1-enter-your-expression.jar - сборка, позволяющая ввести любое регулярное выражение
Нельзя использовать пробелы в выражении.
Пример запуска:
java -jar .\lab1-enter-your-expression.jar 1*1+0
1*1+0 - выражение для примера
